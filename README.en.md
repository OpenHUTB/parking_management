# Iparking停车收费管理系统

#### Description
基于Springboot开发的停车收费管理系统，支持微信支付宝在线支付，支持公众号包月，支持SaaS模式，可以同时支持多个停车场和租户，支持多种硬件设备接入，支持嵌入到Linux中部署厂端和windows部分厂端
